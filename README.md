# Bono Region Youth Parliament Ghana Website

A comprehensive website for the Bono Region Youth Parliament, designed to promote youth development, showcase leaders and initiatives, and facilitate youth engagement in the Bono Region of Ghana.

## 📋 Overview

This website serves as the digital platform for the Bono Region Youth Parliament, providing information about the organization, its leadership, projects, news, and opportunities for youth to get involved.

## 🚀 Features

### Frontend Pages
- **Welcome/Splash Page** - Eye-catching entry with dual logos and region colors
- **Home Page** - Hero section, vision/mission, latest news, upcoming events
- **About Us** - History of Bono Region, Youth Parliament info, values, objectives
- **Leadership** - Profiles of patrons and executive members
- **Projects & Initiatives** - Showcase of community projects with impact stats
- **News & Events** - Latest announcements and calendar
- **Gallery** - Photo gallery with category filters
- **Join Us** - Membership application form
- **Contact** - Contact information, map, and form
- **Production Team** - Behind-the-scenes team profiles

### Admin Panel
- Secure login system
- Dashboard with statistics
- Manage news articles (CRUD)
- Manage projects (CRUD)
- Manage leaders (CRUD)
- View/manage member applications
- Upload gallery images
- Manage events
- View contact messages
- Settings configuration

### Database Structure
- 9 tables: admins, leaders, projects, news, members, gallery, contacts, events, settings
- Proper relationships and indexing
- Timestamp tracking for all entries

## 🎨 Color Scheme

- **Green (#00843E)** - Development and growth
- **Gold (#FCD116)** - Rich culture and prosperity
- **Red (#CE1126)** - National identity and passion
- **White (#FFFFFF)** - Peace and transparency

## 💻 Technologies Used

- HTML5
- CSS3 (with CSS variables, Flexbox, Grid)
- JavaScript (ES6+)
- Font Awesome 6 for icons
- Google Fonts (Poppins)
- MySQL database (XAMPP compatible)
